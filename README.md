# Calculator 🧮

A simple Python calculator program that performs basic arithmetic operations interactively.  
This project is part of the **100 Days of Code: The Complete Python Pro Bootcamp**.

---

## 📖 Overview
The calculator allows users to perform **addition, subtraction, multiplication, and division**.  
It demonstrates the use of functions, loops, and user input handling in Python.

---

## ⚙️ Features
- Supports `+`, `-`, `*`, and `/` operations.
- Interactive prompts for numbers and operations.
- Loops until the user chooses to exit.
- Clear and beginner-friendly structure.

---

## 🚀 Usage
Run the program in your terminal or IDE:

```bash
python main.py
```

You’ll be prompted to:
1. Enter the first number.
2. Enter the second number.
3. Choose an operation (`+ - * /`).
4. View the result.
5. Decide whether to continue or exit.

---

## 🛠️ Example Run
```
Welcome to Calculator
Enter first number: 10
Enter second number: 5
Enter operation: + - * /: *
The result is 50
Do you want to continue? (y/n): n
```
